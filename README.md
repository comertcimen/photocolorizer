# Photo Colorizer

Photo Colorizer is a React app to colorize black and white photos

## Installation
Get an API key from [DeepAI](http://deepai.org)

Put your API key in src/App.js

```node
npm install
```

## Usage

```node
npm start
```

## Build

```node
npm run build
```
